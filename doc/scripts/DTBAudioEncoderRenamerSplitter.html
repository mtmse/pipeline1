Overview

This script will convert wave files in a DTB to MP3, apply renaming to the DTB, and then split the DTB into multiple volumes if needed.

Output

A DTB in one or several volumes, depending on the set maximum volume size, and the sixe of the DTB.

Configuration

This script is a combination of other scrips. For configuration details, please refer to:

  • DTB Audio Encoder
  • Fileset Renamer
  • DTB Splitter
