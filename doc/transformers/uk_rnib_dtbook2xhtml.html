Transformer Purpose

Convert a DTBook document to XHTML, optionally making the XHTML suitable for Daisy 2.02 DTB production.

Note: this transformer uses the XSLT dtbook2xhtml.xsl placed in org/daisy/util/xml/xslt/stylesheets.

Input Requirements

A DTBook document, valid to 2005-1 dtd or later. The document may contain MathML islands as defined by the MathML-in-DAISY specification.

Output

An XHTML 1.0 document. If the input DTBook document contains MathML, the output will be using the W3C DTD XHTML 1.1 plus MathML 2.0 plus SVG 1.1 profile.

On success

A valid XHTML 1.0 or XHTML 1.1 document.

On error

On error, this transformer will send a fatal message, then throw an exception and abort.
