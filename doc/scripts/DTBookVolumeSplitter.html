Overview

This script inserts split points into a DTBook XML document. These points can be used by other scripts such as the DTBookToLaTeX for example, to generate output that is split into multiple volumes.

Input Requirements

A DTBook XML document. Both the input and the output are validated.

Output

A DTBook document. The output is automatically validated, so watch out for error reports.

Configuration

Input DTBook XML file
Required. Path to input DTBook document
Output DTBook XML file
Required. Path to a file to store the output DTBook file in
Number of volumes
Required. Specify in how many volumes the DTBooks should be split into.
Allowed Stretch
Optional. Specify how much a volume can be streched so that the volume splits occur at a nearby level1.
