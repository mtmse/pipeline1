Overview

This script converts a DTBook document to RTF.

Configuration

Input DTBook
Required. The input DTBoook document; DTBook versions supported are 1.1.0, 2002 and 2005.
Include TOC
Required. A boolean indicating if a TOC should be generated
Output RTF
Required. Resulting RTF document.
Include Page Numbering
Optional. A boolean indicating if page numbers should be written to the output.
