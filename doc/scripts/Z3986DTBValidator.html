Overview

This script validates a Daisy/NISO DTB.

The script makes use of the formal Z3986 conformance validator ZedVal. Versions Z39.86-2005 and Z39.86-2002 are supported.

Input Requirements

The Package (opf) file of the DTB to be validated.

Output

By default, this script does not render any physical output; instead, messages are displayed in the user interface. However, by configuring the XML Report setting, you can have the validation result stored in an XML document.

Configuration

Input OPF
Required. Path to OPF file of input DTB
XML Report
Optional. Path to a file to store an XML report in
