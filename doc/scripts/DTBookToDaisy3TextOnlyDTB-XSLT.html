Overview

This script takes a Z39.86-2005 textual content file (DTBook file) and creates a full Z39.86-2005 (DAISY 3) text-only DTB from it.

  • The generated DAISY book will have a spine composed of only one big SMIL document.
  • The conversion does not support DTBook documents with embedded MathML.

Configuration

input
The full path and name of the input DTBook file (Z39.86-2005 textual content file).
outputPath
The directory in which to place the output DTB fileset.
dtbookFix
Selects whether to apply DTBook Fix routines.
