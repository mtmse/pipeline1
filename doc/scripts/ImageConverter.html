Overview

This script converts images associated with a dtbook or xhtml document into jpeg or png images.

Input Requirements

A dtbook or xhtml document containing resolvable image references.

Output

A modified dtbook or html document containing updated references together with the linked images and any additional resources found in the folder containing the source document.
