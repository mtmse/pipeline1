Transformer Purpose

Add @id attributes to elements where it is missing

Input Requirements

A valid DTBook 2002 or DTBook 2005 series document.

Output

An DTBook document (with @id attributes on all relevant elements).

Configuration/Customization

Parameters (tdf)

xml
Path to input DTBook document.
xslt
Stylesheet to use. Defaults to add_ids_to_dtbook.xsl
factory
JAXP factory to use to create an XSLT Transformer instance
out
Path to output DTBook document
