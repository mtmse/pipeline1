Overview

This script will convert wave files in a DTB to MP3.

Configuration

Input file
Required. The manifest member of the input fileset (NCC, OPF, DTBook file, etc)
Output directory
Required. Directory to store the result in. This can not be same directory as the input directory.
Bitrate
Optional. Select the bitrate of the encoded files. Default is 32.
Stereo
Optional. Select wether the files should be encoded in stereo or in mono. Default is mono.
Sampling Rate
Optional. Select the sampling rate of the encoded files. Default is 22050.
