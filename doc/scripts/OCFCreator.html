Overview

Create a Open eBook Publication Structure Container Format (OCF) 1.0 archive from one or several input documents.

Configuration

Input file(s)
Select the document(s)/publication(s) to add to the OCF archive. If you are adding a publication that consists of several files (such as an OEBPS OPS publication), point only to the root (package) file; all other associated files will be added automatically.
Output OCF
Identify the output OCF file. The recommended file extension is epub.
