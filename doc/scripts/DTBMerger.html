Overview

This script merges a Multi Volume DTB into a Single Volume DTB.

The script supports Daisy 2.02, DAISY/NISO 2002 and 2005 DTBs.

Input Requirements

The input fileset main directory. Under this directory, each subvolume to be merged exist in a subdirectory of its own.

Output

A single volume DTB. It is recommended that you validate the output DTB before commencing with further processing.

Configuration

Input directory
Required. Select the input fileset main directory, under which the DTB subvolumes exist in subdirectories.
Output directory
Required. Select the directory where the output DTB will be saved.
Keep redundant files
Select whether redundant files such as prompt files or distInfo.dinf should be copied to output.
