Overview

This script validates a single volume Daisy 2.02 DTB. The validation is not performed by the formal Daisy 2.02 DTB conformance validator. This validator performs a "light mode" validation of the DTB, and can be used in circumstances where full conformance checking is not necessary.

(Note however that the validation that this task does actually performs more tests than the "Light Mode" of the formal Daisy 2.02 DTB conformance validator.)

Input Requirements

The NCC file of the single volume DTB to be validated.

Output

By default, this script does not render any physical output; instead, messages are displayed in the user interface. However, by configuring the XML Report setting, you can have the validation result stored in an XML document.

Configuration

Input NCC
Required. Path to NCC file of input DTB
XML Report
Optional. Path to a file to store an XML report in
