Overview

This script creates a Multi Volume DTB from a Single Volume DTB.

The script supports Daisy 2.02, DAISY/NISO 2002 and 2005 DTBs.

Input Requirements

The input fileset manifest file (NCC or OPF)

Output

A DTB that may or may not have been split into several volumes. If you select a maximum volume size that is larger than the total size of the DTB, the DTB will copied unaltered to the output directory.

If the DTB is indeed split, the DTB volumes will be put in subdirectories of the selected output directory.

Configuration

Input file
Required. Select the input manifest file (NCC or OPF)
Output directory
Required. Select the directory where the output subvolumes will be saved.
Volume size
Optional. Select the maximum size of output volumes, in megabytes. If the destination media is a CD-R, values would typically be in the range between 650 and 700. Default is 680.
Split heading depth
Optional. Select the maximal header level in the navigation control document (ncc/ncx) at which splitting shall be allowed. Default is 2.
