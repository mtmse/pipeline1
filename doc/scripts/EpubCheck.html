Overview

This script validates an IDPF OPS 2.0 publication bundled in an OCF archive (*.epub).

The script makes use of the IDPF EpubCheck tool.

Input Requirements

The OCF (epub) file of the OPS publication to be validated.

Output

By default, this script does not render any physical output; instead, messages are displayed in the user interface. However, by configuring the XML Report setting, you can have the validation result stored in an XML document.

Configuration

Input OPF
Required. Path to Epub file
XML Report
Optional. Path to a file to store an XML report in
